CMD Admin Prompt

This script sets the color of the admin prompt to red:

@echo off
net session >nul 2>&1
if %errorLevel% == 0 (
    PROMPT $E[1;31m$P#$E[0m 
)

The net session command sets errorlevel=2 if executed as non-admin user. This condition is used to set the color of the prompt to E[1;31m (red). At the end, the color is reset to E[0m (white).

The path the the script has to be to set in the registry path Computer\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Command Processor, AutoRun=c:\tools\adminPromptString.bat.
