ForFiles

Examples

forfiles /s /m *.* /d -10 /c "cmd /c echo @path"

Searches recursively (/s) for files matching (/m) the pattern modified (/d) 10 days ago or earlier, then execute a command (/c) to print the path.

forfiles /s /m *.* /d +17/10/2025 /c "cmd /c echo @path"

Searches recursively (/s) for files matching (/m) the pattern modified (/d) after the specified date, then execute a command (/c) to print the path.
